# Study Screen Plugins

Study Screen plugins are for alternate study behaviors and study-session add-ons.

## Possible plugin ideas

- Reverse card mode
- Typing answer mode
- Image prompt mode
- Audio-only mode
- Timer mode
- Streak counter
- Custom answer feedback
- Custom media renderers

## Proposed hook types

- `onSessionStart`
- `onCardRender`
- `onAnswerSelected`
- `onAnswerChecked`
- `onCardComplete`
- `onSessionComplete`
- `studyModes`
- `mediaRenderers`
